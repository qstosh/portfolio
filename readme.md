## WTF?

this is a fork of [ngbp](https://github.com/ngbp/ngbp) (formerly ngBoilerplate).

## Installation

```sh
$ git clone git@github.com:qstosh/portfolio.git
$ cd portfolio
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt watch
```
